### 📱: 기본적인 세팅 끝! Flutter에서 사용하는 레이아웃 구성법을 알아보자

#### 폰에서 흔히 보는 '위젯'을 생각하자

#### 덩어리들을 쪼개자

세로선을 기준으로 가로 덩어리들이 나눠졌으면,
가로 덩어리들을 다시 쪼개자

#### 덩어리 안에 들어가는 재료들

- text
- list (image들의 list, ..)
- button
- stack of containter (그래프 표현 시, .. )
- time picker
- ..

#### ✋🏻 잠깐, 여러 위젯들이 한 줄에 다다닥?

세로축으로 나열된 경우 -> Column
가로축으로 나열된 경우 -> Row

> 이렇듯, 위젯 트리는 쪼개고, 쪼개고, 쪼개고, ..
